TaskFlow AI – Collaborative Kanban SaaS

 🌐 Live Demo

👉 https://taskflow-ai-ten.vercel.app/

⚠️ Note: Backend may take 30–60 seconds to start (free hosting on Render)

 Features:
-  Authentication (JWT-based)
-  Workspace & Board Management
-  Task Creation, Assignment & Status Tracking
-  Drag-and-Drop Task Management (Kanban)
-  Role-Based Access Control (Owner / Member)
-  Real-Time Updates (Socket.IO)
-  AI Task Summarization (OpenAI)
-  File Attachments
-  Activity Timeline (Audit Logs)
-  Modern Responsive UI (Tailwind CSS)

Tech Stack:
### Frontend
- React (TypeScript)
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- Prisma ORM

### Database
- PostgreSQL (Neon)

### Real-time
- Socket.IO

### AI
- OpenAI API

### Deployment
- Vercel (Frontend)
- Render (Backend)

## 🧩 Architecture
Frontend (Vercel) → Backend API (Render) → Database (Neon)
